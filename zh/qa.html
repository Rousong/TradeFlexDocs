快速入门

如何创建我的第一笔交易记录？

创建您的第一笔交易记录：

  1. 打开 TradeFlex 并点击 "+" 按钮
  2. 选择 "添加交易"
  3. 选择资产类型（股票、加密货币、外汇等）
  4. 输入交易详情（代码、数量、价格、日期）
  5. 选择是买入还是卖出交易
  6. 点击 "保存" 记录交易
提示： 如果您有现有记录，也可以从CSV文件导入交易。
跟踪交易需要哪些信息？

为了准确跟踪，您应该记录：

必需信息：
  • 资产代码（如 AAPL、BTC）
  • 交易类型（买入/卖出）
  • 数量/交易量
  • 单价
  • 交易日期和时间
可选信息：
  • 交易费用/佣金
  • 备注或使用的策略
  • 投资组合/账户名称
  • 分类标签

交易记录管理

如何从券商导入交易记录？

您可以使用CSV文件导入交易：

  1. 从您的券商导出交易历史为CSV文件
  2. 在 TradeFlex 中，前往 设置 → 导入/导出
  3. 点击 "导入CSV"
  4. 选择您的CSV文件并将列映射到TradeFlex字段
  5. 预览并确认导入
注意： 确保您的CSV文件包含所有必需字段（代码、类型、数量、价格、日期）。
添加后可以编辑或删除交易吗？

是的，您可以随时修改您的交易：

编辑交易：
  1. 在交易列表中找到交易
  2. 点击交易打开详情
  3. 点击 "编辑" 按钮
  4. 进行更改并保存
删除交易：
  1. 打开交易详情
  2. 点击 "删除" 按钮
  3. 确认删除
注意：除非您有备份，否则已删除的交易无法恢复。

分析与报告

盈亏是如何计算的？

TradeFlex 使用先进先出（FIFO）方法计算盈亏：

示例：
  • 买入： 100股苹果股票 @ $150 = $15,000
  • 卖出： 50股苹果股票 @ $160 = $8,000
  • 利润： (160 - 150) × 50 = $500

应用自动考虑：

  • 交易费用和佣金
  • 货币转换
  • 股票分割和股息
  • 洗售规则（如适用）
可以导出税务报告吗？

是的，TradeFlex 提供多种税务报告导出选项：

可用格式：
  • PDF报告
  • Excel电子表格
  • CSV文件
  • 税务专用模板
报告类型：
  • 已实现收益/损失
  • 未实现持仓
  • 股息收入
  • 交易摘要
重要提示： 对于具体的税务建议，请务必咨询税务专业人士。

故障排除

应用运行缓慢或崩溃

尝试以下故障排除步骤：

快速修复：
  1. 强制关闭并重启应用
  2. 重启您的设备
  3. 检查应用更新
  4. 释放设备存储空间
高级解决方案：
  1. 清除应用缓存（Android）
  2. 重新安装应用
  3. 检查设备兼容性
  4. 联系支持并提供崩溃日志
重新安装前： 请确保备份您的数据，以避免丢失交易记录。
如何备份和恢复我的数据？

通过定期备份保护您的交易数据：

创建备份：
  1. 前往 设置 → 备份与恢复
  2. 点击 "创建备份"
  3. 选择备份位置（本地/云端）
  4. 等待备份完成
恢复数据：
  1. 前往 设置 → 备份与恢复
  2. 点击 "从备份恢复"
  3. 选择您的备份文件
  4. 确认恢复操作
建议： 设置自动每日备份以确保您的数据始终受到保护。
